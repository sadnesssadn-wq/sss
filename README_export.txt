║          自动提取当天所有EMS运单号                           ║
╚══════════════════════════════════════════════════════════════╝

📦 脚本: export_today_all.py

🚀 使用方法:
   python3 export_today_all.py

✨ 功能:
   ✅ 自动获取当前日期
   ✅ 统计当天运单号数量（去重）
   ✅ 批量导出所有运单号
   ✅ 显示实时进度和剩余时间
   ✅ 自动保存到 tracking_YYYYMMDD.txt

📊 输出格式:
   tracking_20251011.txt
   
   内容格式（每行一个）:
   EA044826727VN
   EG044557336VN
   ES072634095VN
   ...

⏱️  速度:
   约 6-10个运单号/分钟
   8500个运单约需 14-20分钟

💡 提示:
   - 脚本会自动显示进度和剩余时间
   - 可以随时Ctrl+C中断（已提取的会保存）
   - 每天运行一次即可获取最新数据
