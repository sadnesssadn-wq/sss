# 📁 项目文件说明

## 🎯 核心脚本文件

### 1. `scan_today_full_api.py` ⭐ **正式版扫描工具**
- **用途**: 大规模扫描今天的未签收订单
- **特点**: 
  - 同时调用 3 个 API
  - 获取 42+ 个完整字段
  - 目标 10,000 个订单
  - 30 线程并发
  - 100 个代理池
- **运行**: `python3 scan_today_full_api.py`
- **耗时**: 30-60 分钟

---

### 2. `test_scan_full_api.py` 🧪 **测试版工具**
- **用途**: 快速测试功能是否正常
- **特点**: 
  - 小范围扫描（100个号段）
  - 只找 10 个订单
  - 详细显示每个 API 调用过程
  - 5 个代理
- **运行**: `python3 test_scan_full_api.py`
- **耗时**: 1-2 分钟
- **建议**: ⭐ **首次使用请先运行这个！**

## 🎯 获取的数据

### 三个 API 提供的信息

#### API 1: Inquiry（38个字段）
- 发件人/收件人完整信息
- 所有金额明细（7个字段）
- 日期信息（3个字段）
- 配送信息（6个字段）
- 配送证据（照片URL）
- 其他信息（10个字段）

#### API 2: DeliveryLadingJourney
- 配送轨迹列表
- 配送历史记录
- 是否已签收

#### API 3: Gateway/Bussiness
- 商品列表
- 商品名称、数量、重量
- 单价、总金额

**总计: 42+ 个字段**
