# sss

这是一个新项目。

## 开始使用

欢迎使用此项目！

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python main.py
```

## 运行测试

本项目包含单元测试。运行测试：

```bash
# 运行所有测试
python -m pytest test_main.py

# 运行测试并显示详细信息
python -m pytest test_main.py -v

# 运行测试并生成覆盖率报告
python -m pytest test_main.py --cov=main --cov-report=html
```

或者使用 Python 内置的 unittest：

```bash
python test_main.py
```
