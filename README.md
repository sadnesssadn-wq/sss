# 检查用户疲劳状态 Check If User Is Tired

一个简单友好的小程序，用来关心用户是否疲倦，并给出健康建议。

A simple and friendly program to check if the user is tired and provide health suggestions.

## 功能 Features

- 🇨🇳 中文界面支持
- 😴 智能识别用户的疲劳状态
- 💡 提供实用的休息建议
- 👋 友好的交互体验

## 使用方法 Usage

### 运行程序 Run the program

```bash
python3 check_if_tired.py
```

### 示例对话 Example conversation

```
你好呀！👋
让我来关心一下你~

你困不困？(是/否 或 yes/no): 是

😴 看来你确实有点累了呢！
💡 建议：
   • 休息一下，喝杯水
   • 站起来活动活动
   • 如果可以的话，小睡15-20分钟
   • 保持良好的作息很重要哦！
```

## 支持的回答 Supported responses

**表示疲倦 Tired:**
- 是 / yes / y
- 困 / 累 / 疲倦 / 想睡

**表示不累 Not tired:**
- 否 / no / n
- 不困 / 不累 / 精神

## 健康小贴士 Health Tips

记得：
- ⏰ 保持规律作息
- 💧 多喝水
- 🏃 适当运动
- 😴 充足睡眠
