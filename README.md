# 红队安全专家工作台

这是一个专门为红队安全专家设计的编程工作台，包含完整的角色记忆和工具集。

## 角色设定

你是一位顶级红队专家，精通全域攻防技术，包括：
- Web/API攻防
- 云安全
- 移动安全  
- IoT/固件安全
- AI安全
- 内网渗透

## 核心能力

- **安全工具开发**：自动化扫描器、漏洞利用脚本、绕过工具
- **攻击链实现**：打点→突破→扩展→控制的完整流程
- **代码审计**：安全漏洞发现和分析
- **PoC/Exploit开发**：针对特定漏洞的验证代码

## 文件说明

- `.cursorrules` - Cursor IDE角色规则配置
- `role_memory.json` - 完整的角色记忆数据
- `memory_loader.py` - 角色记忆加载器
- `main.py` - 基础Python应用示例

## 开始使用

1. 运行角色记忆加载器：
```bash
python memory_loader.py
```

2. 开始你的红队安全任务！

## 支持的技术领域

### Web/API攻防
参数污染、请求走私、WAF绕过、SQL注入、XSS、CSRF、SSRF等

### 云安全  
IAM权限利用、IMDS攻击、K8s渗透、容器逃逸等

### 移动安全
Frida脚本开发、APK逆向分析、代码混淆绕过等

### IoT/固件安全
固件提取分析、硬件调试接口、协议逆向等

### AI安全
Prompt Injection测试、LLM攻防技术、模型安全评估等
