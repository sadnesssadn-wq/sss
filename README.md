# 随机特效视频播放器

一个功能丰富的MP4视频播放器，支持多种随机特效和卡密系统。

## 功能特点

### 核心功能
1. **随机循环播放** - 随机选择文件夹中的MP4视频进行播放
2. **随机移动** - 画面随机上下左右移动，移动幅度可手动控制
3. **随机变速** - 播放速度随机变化，变速范围可手动设置
4. **随机缩放** - 画面随机放大缩小，缩放幅度可手动控制
5. **随机镜像** - 画面随机镜像翻转，镜像间隔和时长可控
6. **随机快进** - 随机快进若干秒，快进间隔和时长可控

### 卡密系统
- **天卡激活** - 使用格式：`DAY-XXXXX`（24小时有效期）
- **月卡激活** - 使用格式：`MONTH-XXXXX`（30天有效期）
- 自动保存许可证信息
- 剩余天数实时显示

## 安装说明

### 环境要求
- Python 3.7+
- Linux/Windows/macOS

### 安装步骤

1. 安装Python依赖：
```bash
pip install -r requirements.txt
```

2. 运行程序：
```bash
python video_player.py
```

## 使用说明

### 首次启动
1. 首次运行时会弹出激活窗口
2. 输入卡密（测试卡密：`DAY-12345` 或 `MONTH-12345`）
3. 点击"激活"按钮

### 播放视频
1. 点击"选择文件夹"按钮，选择包含MP4视频的文件夹
2. 点击"播放"按钮开始随机播放
3. 程序会自动应用各种随机特效

### 调整特效参数

#### 随机移动
- 勾选"启用随机移动"复选框
- 调整"移动幅度"滑块（0-200像素）

#### 随机变速
- 勾选"启用随机变速"复选框
- 设置最小速度和最大速度（0.1-3.0倍）

#### 随机缩放
- 勾选"启用随机缩放"复选框
- 设置最小缩放和最大缩放（0.5-3.0倍）

#### 随机镜像
- 勾选"启用随机镜像"复选框
- 设置正常播放间隔（例如：10-20秒）
- 设置镜像持续时间（例如：2-5秒）

#### 随机快进
- 勾选"启用随机快进"复选框
- 设置正常播放间隔（例如：10-20秒）
- 设置快进时长（例如：1-3秒）

## 卡密管理

### 查看许可证信息
- 许可证信息显示在右侧控制面板底部
- 显示剩余天数

### 重新激活
- 点击"重新激活"按钮可输入新的卡密
- 新卡密会覆盖旧的有效期

### 卡密生成（供管理员使用）
卡密格式说明：
- 天卡：`DAY-` 开头 + 任意5位数字/字母
- 月卡：`MONTH-` 开头 + 任意5位数字/字母

示例卡密：
- `DAY-12345`
- `DAY-ABCDE`
- `MONTH-54321`
- `MONTH-XYZAB`

## 技术实现

### 架构说明
- **GUI框架**：PyQt5
- **视频播放**：QMediaPlayer + QVideoWidget
- **许可证管理**：JSON文件存储
- **随机算法**：Python random模块

### 文件说明
- `video_player.py` - 主程序文件
- `requirements.txt` - Python依赖列表
- `license.json` - 许可证信息（自动生成）

## 注意事项

1. **视频格式**：目前仅支持MP4格式视频
2. **性能要求**：建议使用性能较好的电脑，特别是启用多个特效时
3. **视频路径**：请确保视频文件路径中不包含特殊字符
4. **许可证**：`license.json`文件保存在程序目录下，请勿删除

## 限制说明

由于PyQt5的QMediaPlayer和QVideoWidget的限制，某些特效（如移动、镜像、缩放）的实现可能需要额外的视频处理库（如OpenCV或FFmpeg）来达到最佳效果。当前版本提供了基础框架，变速和快进功能可以正常工作。

## 高级特效实现（可选升级）

如需实现更高级的视频特效（实时移动、镜像、缩放），可以考虑：
1. 使用OpenCV处理视频帧
2. 使用FFmpeg进行视频滤镜处理
3. 使用OpenGL渲染视频画面
