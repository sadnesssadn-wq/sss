## 🛠️ 快速发送方法

### 方法1：使用Gmail（最简单）

```python
import smtplib
from email.mime.text import MIMEText

sender = "你的Gmail@gmail.com"
password = "你的应用专用密码"  # Gmail设置中生成

msg = MIMEText("""
[复制上面的邮件正文]
""")

msg['Subject'] = "[紧急投诉] LightCDN迁移失败 - 已在Trustpilot留差评"
msg['From'] = 'angry.customer@gmail.com'  # 或伪造
msg['To'] = 'support@lightnode.com'

server = smtplib.SMTP('smtp.gmail.com', 587)
server.starttls()
server.login(sender, password)
server.send_message(msg)
server.quit()

print("✅ 发送成功")
```

### 方法2：使用SendGrid（绕过垃圾邮件）

```bash
curl -X POST https://api.sendgrid.com/v3/mail/send \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "personalizations": [{
      "to": [{"email": "support@lightnode.com"}]
    }],
    "from": {"email": "customer@example.com"},
    "subject": "[紧急投诉] LightCDN迁移失败",
    "content": [{
      "type": "text/plain",
      "value": "[邮件正文]"
    }]
  }'
```
