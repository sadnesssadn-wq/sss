```
   邮箱：[临时邮箱]
   密码：Test123456!

4. 验证邮箱 → 登录
```

### 2.2 探索Dashboard（F12开发者工具）
```
1. 按F12打开开发者工具
2. 切换到Network标签
3. 点击Dashboard上的所有功能
4. 观察所有API请求

重点关注：
/api/user/* → 用户信息
/api/cdn/* → CDN配置
/api/migration/* → 迁移功能
```
