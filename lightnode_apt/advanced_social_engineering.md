# 高级社工攻击方案

## 方案1: 假冒招聘候选人
**目标**: 获取HR邮箱和组织架构

```
邮件标题: Application for DevOps Engineer Position
收件人: support@lightnode.com

内容:
Dear HR Team,

I saw your DevOps Engineer position on LinkedIn.
I'd like to apply, but the job posting doesn't show the HR contact email.

Could you please provide:
1. HR manager's email for direct application
2. Team size (so I can understand the scale)
3. Reporting structure (who this role reports to)

Thank you!
```

## 方案2: 假冒合作媒体
**目标**: 获取CEO/CTO/PR负责人信息

```
邮件标题: Interview Request - Cloud Computing Feature
收件人: business@lightnode.com

内容:
Hello LightNode Team,

We're TechCrunch Asia, preparing a feature on "Rising Cloud Providers in 2025".

We'd like to interview your CEO or CTO about:
- Company vision
- Technical innovations
- Market strategy

Could you provide:
1. CEO/CTO name and email
2. PR/Media contact
3. Brief bio for our article

Looking forward to your response!
```

## 方案3: 假冒大客户

## 电话社工话术

### 场景1: 打给总机 (852-56462884)
```
你好，我是XX公司的，想找你们的：
- "HR部门" → 获取HR分机/邮箱
- "技术总监" → 获取CTO分机/邮箱
- "市场部" → 获取市场负责人信息
- "XX先生/女士"（假装认识）→ 前台可能透露此人是否存在

技巧：
- 表现得很熟悉公司，降低戒心
- 说"之前联系过，但忘记分机了"
- 问"XX部门现在还是张经理负责吗？"（套出真实姓名）
```

### 场景2: 打给客服
```
"你好，我想投诉/表扬你们的员工，但不知道他的全名。
他姓张，在技术支持部门，大概30多岁。"

→ 客服可能为了处理投诉/表扬，透露员工全名
```

## Salesmartly在线客服话术

### 开场
```
"你好，我们是企业客户，需要咨询大规模部署。
请问有专门的企业销售团队吗？"
```

### 进阶问题
```
1. "你们的销售经理是谁？我想直接和他对接"
2. "你们团队有多少人？客服24小时吗？"
3. "我之前和一位姓李的销售聊过，还在吗？"
4. "技术问题需要找技术部，邮箱是什么？"
5. "HR邮箱是什么？我们想谈校招合作"
```

### 高级套路
```
"我是你们的老客户（假装），但换了邮箱，
之前的客户经理是张XX，现在还在吗？"

→ 如果客服说"没有这个人"，就换个姓继续试
→ 如果客服说"还在"，就追问"那他现在邮箱是什么？"
```
