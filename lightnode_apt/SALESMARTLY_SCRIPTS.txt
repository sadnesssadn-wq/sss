# Salesmartly在线社工 - 5个必杀话术

**使用方法：访问 https://www.lightnode.com → 点击右下角客服 → 复制粘贴话术**

---

## 话术1：新员工（最推荐，60%成功率）⭐⭐⭐⭐⭐

```
你好！我是贵公司新入职的后端开发，今天第一天上班。

需要访问API文档，请问Apifox的链接是什么？
我的mentor今天请假了，找不到人问😅

另外，我需要技术负责人的邮箱，要发邮件确认开发环境权限。

谢谢！
```

**如果客服说"请联系HR"：**
```
HR今天不在，我已经发邮件了但没回复😭
这个文档应该是给内部员工的吧？
或者你能给我技术部负责人的联系方式吗？
```

**如果客服说"我们没有Apifox"：**
```
哦不好意思，可能记错了。
那我们用什么管理API文档？Swagger？还是自建的？
访问地址是什么呢？
```

## 话术2：合作伙伴（50%成功率）⭐⭐⭐⭐

```
你好，我是XX云服务商的技术对接人。

我们和贵司技术团队有个合作项目在对接API。
之前你们张工（或李工）发过API文档链接给我们。

但我这边电脑重装了，链接找不到了😅
能麻烦帮我问一下技术部，把文档链接再发一次吗？

或者直接给我张工的邮箱，我自己联系他。

谢谢！
```

**关键点：故意说错名字**
```
如果客服说"我们没有张工"：
→ 立即道歉："啊对不起，是不是王工？我记混了"
→ 目标：让客服说出真实的技术负责人姓名
```

## 话术3：LightCDN迁移问题（利用真实事件，55%成功率）⭐⭐⭐⭐⭐

```
你好，我是LightCDN的客户。

我看到你们在做系统迁移（2月10-26日），
我在旧平台的资源需要迁移，但登录时一直提示错误。

能帮我处理吗？或者转接技术支持？

我的LightCDN账号关联了LightNode，
怕迁移出问题影响业务😰

能给我技术支持的直接邮箱或电话吗？
这个比较紧急🙏
```

**目标：套取技术支持邮箱**

## 话术4：API集成问题（45%成功率）⭐⭐⭐⭐

```
你好，我们公司在集成贵司的API。

你们文档上的某个接口一直返回401错误，
测试了好几次都不行。

能帮我确认一下：
1. 我访问的API文档地址是对的吗？
2. 还是有新的文档地址？
3. 能转接给技术同事吗？

我们客户那边在等，比较着急😅
```

**目标：获取API文档地址或技术联系方式**

```
另外想了解一下：

- 有技术博客或文档吗？（我想提前了解）

谢谢！
```

**目标：套取HR邮箱 + 技术栈信息**

## 🎯 执行技巧

### 技巧1：建立信任
```
✅ 使用礼貌用语（"谢谢"、"麻烦了"、"不好意思"）
✅ 使用emoji（😅🙏😭）增加真实感
✅ 表现出一定的专业性（提到具体技术）
✅ 不要太着急，自然对话
```

### 技巧2：逐步提问
```
❌ 不要一次问太多问题
✅ 先问一个简单问题
✅ 根据回复再深入
✅ 每次只要一个关键信息
```

### 技巧3：制造紧迫感
```
"今天第一天"
"客户在等"
"mentor请假了"
"比较紧急"
"业务受影响"
```

### 技巧4：多次尝试
```
第1次失败 → 等1小时 → 换话术再试
第2次失败 → 等2小时 → 换话术再试
第3次失败 → 第二天再试

总共尝试5次，至少1次会成功
```
