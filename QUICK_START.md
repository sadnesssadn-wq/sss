## 💡 支持的格式

### ✅ 基本格式
```
EP492966929VN
EP492940379VN
```

### ✅ 带注释
```
# 这是注释，会被忽略
EP492966929VN

# 可以添加说明
EP492940379VN
```

### ✅ 带空行
```
EP492966929VN

EP492940379VN

EP493183769VN
```

### ✅ 混合格式
```
# ===== 客户A =====
EP492966929VN
EP492940379VN

# ===== 客户B =====
EP493183769VN

# 空行会被忽略

EL135248100VN
```
