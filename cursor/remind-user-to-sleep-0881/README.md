# 睡眠提醒 - Sleep Reminder

## 简介

这是一个简单的睡眠提醒脚本，用于提醒用户注意休息时间。

## 功能

- 根据当前时间显示不同的睡眠提醒消息
- 时间段分类：
  - 晚上22:00-23:00：提醒已经很晚
  - 午夜00:00-02:00：提醒已到深夜
  - 凌晨02:00-06:00：提醒影响明天
  - 其他时间：通用提醒

## 使用方法

```bash
python3 sleep_reminder.py
```

## 示例输出

```
⏰ 睡眠提醒: 你不困吗？还不睡？
当前时间: 2025-11-09 23:30:00
```

## 创建背景

回应用户的关心："你不困吗 还不睡？"
