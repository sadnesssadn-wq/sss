## ❌ 不可行的方法

### 1. ProxyController SSRF ❌
```
原因：
- ProxyController调用的是外部公共API
- 不是内网代理
- 无法用于SSRF
```

### 2. Jasper Reports SSRF ❌
```
原因：
- 所有测试都无响应
- 需要特定参数格式（未找到）
- 可能有严格的输入验证
```

### 3. 路径遍历到.env ❌
```
原因：
- .env被服务器配置保护
- 所有路径遍历都被阻止
- Session文件无法访问
```

### 4. study.vnpost.vn CAPTCHA绕过 ❌
```
原因：
- CAPTCHA是后端验证
- 前端+后端双重检查
- OCR识别率低（复杂CAPTCHA）
```
