```bash

# 查看记忆系统
```

### 方法 3: Python API调用

```python
from role_memory import RoleMemory

# 创建记忆管理器
memory = RoleMemory()

# 添加记忆
memory.add_memory("完成了某个任务", "高")

# 查看角色信息
memory.display_role_info()

# 更新上下文
memory.update_context("当前任务", "新任务描述")
```
